## üöÄ Usage Tips

1. **PDF Files**: Open any `.pdf` file and it will display in the editor
2. **CSV Files**: Open `.csv` files - they'll be colorized and you can edit them in table view
3. **Markdown**: Press `Cmd+Shift+V` (Mac) or `Ctrl+Shift+V` (Windows) to preview markdown files
4. **Images**: Simply open image files - they'll display automatically
5. **YAML/JSON**: Syntax highlighting and validation work automatically
